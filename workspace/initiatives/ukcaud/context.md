# UK Cloud Audit (UKCAUD) Context

## Overview

UK Cloud Audit delivery project (Jira: UKCAUD). Primary delivery board for stories, bugs, epics, and initiatives.

## Key Stakeholders

- Product Management — strategic alignment
- Sales — customer requirements
- Training — adoption/enablement
- Support — issue escalation

## Ticket Patterns

- **Stories:** Content + Peer Review subtasks, product-specific components
- **Bugs (DEVELOP):** UKCAUD_BUGS_DEV/QA + UKCAUD_DEVELOP labels, Development + Manual Testing subtasks
- **Bugs (RELEASE):** UKCAUD_BUGS_DEV/QA + UKCAUD_RELEASE labels, hotfix fix versions
- **Naming:** `{DEVELOP|RELEASE} - {Product} - {Bug Description}`
